Data Flow
PDF upload ──► /match-cv (file)
                  │
                  ▼
          pdf_service.py ──► extract_text_from_pdf()
                  │
                  ▼
          OllamaModelClient.chat() ──► CV analysis prompt
                  │
                  ▼
          profile_parser.py ──► parse_cv_profile()
                  │
                  ▼
          database_service.search_jobs() ──► PostgreSQL
                  │
                  ▼
          ranked_jobs (JSON) ──► top 10 with match_ratio%
Q&A flow:
  /cv-qa/init (file) ──► pdf_service ──► qa_service.init_session()
                                              │
  /cv-qa/answer (session_id + answer) ◄──── question
                                              │
                                        is_complete? ──► profile JSON
  /match-from-profile (profile JSON) ──► database_service ──► ranked_jobs
API Endpoints
Method	Endpoint	Input	Output
GET	/health	—	{"status": "ok"}
POST	/match-cv	PDF file (multipart)	CV profile + ranked jobs
POST	/cv-qa/init	PDF file	session_id + first question
POST	/cv-qa/answer	session_id + answer	next question OR final profile
GET	/cv-qa/{id}	—	session state
POST	/match-from-profile	profile JSON	ranked jobs
